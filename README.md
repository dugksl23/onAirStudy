# OnAirStudy
각 카테고리별로 스터디방을 생성하여 스터디원을 모집,활동 할 수 있게 도움을 주는 온라인 스터디 웹사이트
 
 **스터디방 기능**
* 방장 - 방정보 수정
* 방별 스터디원 단체 채팅
* 시험문제등록 , 시험보기, 채점
* 출석체크
* 스터디방 공통 캘린더,스케줄러
